## **Prime Number Checker**
### Objectifs pédagogiques
* Manipuler les nombres et appliquer des algorithmes de base.
* Gérer des cas simples puis complexes.
* Pratiquer les boucles, les conditions et l'optimisation des algorithmes.

### Énoncé
Écrire une fonction `isPrime` qui prend un **nombre entier** et retourne `true` si le nombre est premier, sinon `false`.

#### Règles
1. Implémenter une fonction `isPrime(int number)` qui retourne `true` si le nombre est premier, sinon `false`.
2. Optimiser l'algorithme pour qu'il soit plus rapide qu'une simple recherche de tous les diviseurs jusqu’au nombre (`O(n)`).
3. Voici quelques exemples :
    * Nombre premier (par exemple, 5, 7, 13).
    * Nombre non premier (par exemple, 4, 6, 8).
    * Cas limites (par exemple, 0, 1, 2).
